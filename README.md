# devel

LaughTrack is a mobile-first, cinematic comedy community UI/UX prototype that blends:
- Reddit-style interaction patterns (threaded discussions, upvote/downvote, communities, infinite feed)
- Comedy magazine editorial energy (bold headlines, satirical tone, featured stories)
- Mockumentary cinematic flair (freeze-frame feel, grain overlays, dramatic punchy motion)

## Run
Open `/home/runner/work/devel/devel/index.html` in a browser.

## Included core screens and behaviors
- Home feed with featured cinematic hero and rotating comedic headlines
- Post scene dialog with threaded comments and reaction bar
- Create post flow (text, meme-style, poll type selector) with punchline helper
- User profile panel with reputation badge and comedy stats
- Community/category navigation and discovery search
- Playful notifications with humorous microcopy
- Dark mode toggle + cinematic mode toggle
- Responsive layout and accessible labels/controls
